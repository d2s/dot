# Ruby-based tools

## Tools

### 3rd party tools

- [ttscoff/doing](https://github.com/ttscoff/doing)
  - "A command line tool for remembering what you were doing and tracking what you've done."


## Installing

### Install dependencies

```sh
gem install bundler
```

### Install tools

```sh
bundle install
```
